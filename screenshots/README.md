# Screenshots

This directory contains UI screenshots used in the main README.

## Required Screenshots

The following images are referenced in `README.md` and should be captured once the dashboard and CLI are functional:

| Filename | Description | Dimensions |
|---|---|---|
| `banner.png` | Project banner / hero image | 1280×640 |
| `dashboard.png` | Live trading dashboard overview | 1280×800 |
| `grid_view.png` | Grid visualization with filled/open levels | 1280×800 |
| `backtest.png` | Backtest results report | 1280×800 |
| `portfolio.png` | Portfolio analytics view | 1280×800 |
| `sentiment.png` | Market sentiment panel | 1280×800 |
| `risk.png` | Risk monitor panel | 1280×800 |

## Guidelines

- Use light mode screenshots for consistency
- Capture at 2x resolution (Retina) where possible and scale down in HTML
- Redact any real API keys, account balances, or personally identifiable information
- Use paper trading / testnet data for all demo screenshots
